Висновок: Під час виконання лабораторної роботи були закріплені навички роботи з директивами Dockerfile і запуском багатосервісних застосуноків за допомогою Docker Compose. Вули виявлені і виправлені декілька помилок, які не дозволли запускатися одному з крнтейнерів. Головною помилкою було exec format error, і в решті решт вона була вирішена зміною версії python. Для вирішення цієї та декількох інших невеликих проблем і запуску контейнерів з подальшим відкриттям сторінки було витрачено 9 годин.
